    • ¿Qué es una red informática?


    Es un conjunto de dispositivos electrónicos interconectados que comparten recursos y permiten la comunicación entre sí.Estos dispositivos incluyen computadoras, servidores, enrutadores, y otros equipos que se conectan mediante cables o tecnologías inalámbricas.

    Qué es una red LAN.


    Una red LAN (Local Area Network o Red de Área local) es un conjunto de dispositivos informáticos interconectados dentro de un área geográfica limitada, como un edificio

    Qué es una red WAN.


    Una red WAN (Wide Area Network) es una infraestructura de comunicación que conecta dispositivos a través de una amplia área geográfica, como ciudades, países o incluso continentes.

    Qué es Internet.


    Es una red lobal de computadoras interconectadas que se comunican entre sí mediante protocolos estándar. Permite la transferencia de datos y el acceso a recursos, comopáginas web, correos electrónicos y otros servicios en línea.

    Qué es la Web.


    Como adjetivo significa 'de la Red o de Internet' . Se usa normalmente en la expresión página web, que significa 'documento de Internet, al que se accede mediante enlaces de hipertexto.

    Qué es WWW


    La World Wide Web (WWW) es un sistema de información en el que documentos multimedia están vinculados a través de enlaces hipertextuales. Estos documentos, generalmente, páginas web, contienen texto, imágenes, vídeos y otros recursos multimedia. Los usuarios acceden a la web mediante navegadores web para explorar contenido, realizar búsquedas e interactuar con servicios en línea.

    Qué es el protocolo HTTP y HTTPS.


    HTTP es un protocolo de transferencia de hipertexto sin cifrado. Se utiliza para la transmisión de datos, como texto, imágenes y otros recursos, entre un navegador web y un servidor. Esta información no está encriptada, lo que la hace vulnerable a manipulación de terceros.
    HTTPS es una versión segura de HTTP que utiliza en protocolo SSL/TLS para cifrar la comunicación entre el navegador y el servidor. Esto proporciona una capa adicional de seguridad, haciendo que sea más difícil para los atacantes interceptar o alterar datos transmitidos.

    Página web estática vs página web dinámica.


    Una página web estática presenta contenido fijo y predefinido. Su información no cambia según la interacción del usuario ni en tiempo real.
    Una página web dinámica se genera en tiempo real y contenido puede cambiar según las interacciones del usuario o las condiciones específicas.

    Ejemplos: wordpress, joomla, wix.

    Proveedor de servicios de Internet. Ejemplos.


    El proveedor de servicios de internet es la empresa que brinda conexión a Internet a sus clientes. . Ejemplo: Amazon Web Services (AWS)

    Dirección IP. Diferencia entre una dirección IP privada y pública.


    Una dirección IP (Protocolo de Internet) es un identificador numérico asignado a cada dispositivo conectado a una red informática que utiliza el protocolo de Internet para la comunicación.
    Una dirección IP privada se utiliza dentro de una red privada para conectarse de forma segura a otros dispositivos dentro de esa misma red. Una dirección IP pública te identifica ante el resto de Internet para que toda la información que buscas pueda encontrarte.

    ¿Qué es el alojamiento web/hosting? Ejemplos.


    Es el servicio que proporciona espacio en servidores conectados a Internet para almacenar y hacer accesibles los archivos y datos de un sitio web. Ejemplo: CMS Hub, Hostinger, SiteGround.
